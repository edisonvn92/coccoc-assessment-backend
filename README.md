# API Backend 
This project is for backend API of Coccoc Assessment

## Version
- NodeJS: 18.14.2
- ExpressJS: 4.18.2

## Install package
- If local NodeJS version is different, install nvm to manage NodeJS version (https://github.com/nvm-sh/nvm)
- Run `npm install` to install packages

## Local Development
Run `npm run start` to create local server at `http://localhost:3000`
